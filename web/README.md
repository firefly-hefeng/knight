# Knight System Frontend

Modern AI Agent Management Dashboard built with Next.js 14 + shadcn/ui.

## Quick Start

```bash
cd web
npm install
npm run dev
```

Open http://localhost:3000

## Features

- **Task Dashboard** - Create and monitor tasks
- **Agent Army** - View agent status and capabilities
- Dark mode by default
- Responsive design

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui components
